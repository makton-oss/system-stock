Features
Semua yang korang perlukan
Direka khas untuk outlet F&B, kedai runcit, dan gudang yang nak sistem simple tapi reliable.
📥

Stock In / Out / Wastage

Staff hantar request terus via WhatsApp. Support multi-item dalam satu mesej.

✅

Approval Flow

Manager atau supervisor approve / reject tekan butang. Anti double-approval built-in.

⚠️

Low Stock Alert

Manager auto notified bila stok bawah minimum. Takde lagi stok habis tiba-tiba.

📊

Laporan Lengkap

Summary, flow, detail, dead stock dan wastage, semua dalam WhatsApp. Pilih bulan, tekan butang.

🏪

Multi Outlet

Satu manager boleh handle banyak outlet. Data diasingkan ikut outlet.

📸

Daily Snapshot

Inventori auto-snapshot setiap tengah malam. Track nilai stok mengikut masa.

Cara kerja
Simple je prosesnya
Dari staff rekod sampai manager approve, semua dalam WhatsApp.

Staff taip command

"OUT ayam 5" atau "IN tepung 10" terus dalam WhatsApp.

Sistem proses & notify

Request masuk pending, manager dapat notifikasi automatik dengan butang approve/reject.

Manager approve

Tekan APPROVE, stok terus update, movement direkod, log tersimpan.

Laporan auto

Taip "REPORT" untuk tengok summary. Pilih bulan, data siap dalam saat.
